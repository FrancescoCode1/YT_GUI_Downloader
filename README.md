# YT_GUI_Downloader
As the name suggests this is a Youtube GUI Downloader based on [yt-dlp](https://github.com/yt-dlp/yt-dlp)

It features 
- Conversion to wav and mp3 (more soon!)
- Filenames = Titles
- Multiple downloads at once

https://streamable.com/gi07pe

## How to use:
- Open it
- Add links
- Choose filetype
- convert
